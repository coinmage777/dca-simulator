# 암호화폐 분할매수(DCA) 시뮬레이터

비트코인, 이더리움, 솔라나, 리플의 과거 월별 원화(KRW) 데이터로 분할매수 수익률을 시뮬레이션하는 웹앱입니다.

## 기능

- **코인 선택**: 비트코인(주황), 이더리움(파랑), 솔라나(초록), 리플(하늘색) — 코인별 아이콘·테마 색상 적용
- **1회 투자금액**: 5만원, 10만원, 20만원, 50만원
- **매수 주기**: 매주, 격주, 매월
- **시작 시점**: 2022년 1월부터 6개월 단위 (2022-01 ~ 2025-01)
- **데이터**: 2022년 1월 ~ 2025년 2월 월별 원화(KRW) 종가 (하드코딩)
- **결과**: 총 투자금액, 현재 평가금액, 수익률, 누적 투자금 vs 평가금액 차트

## 실행 방법

```bash
npm install
npm run dev
```

브라우저에서 http://localhost:5173/ 로 접속합니다.

## 빌드

```bash
npm run build
npm run preview   # 빌드 결과 미리보기
```

## 기술 스택

- React 18 + TypeScript
- Vite
- Recharts (차트)
- 코인별 월별 원화(KRW) 종가 데이터 하드코딩 (2022-01 ~ 2025-02)

## 면책

데이터는 참고용이며, 실제 투자 수익을 보장하지 않습니다.
